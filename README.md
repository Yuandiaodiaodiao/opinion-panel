# BSC 预测市场交易工具

## 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 配置
- 编辑 `config.js` 文件，配置账户地址和合约信息
- 复制 `.env.example` 为 `.env` 并填入私钥和授权信息

### 3. 运行
```bash
npm start [topicId]
```

默认使用 Topic ID 789，可通过命令行参数指定其他 Topic

⚠️ **注意**：请妥善保管你的私钥，不要泄露给任何人
